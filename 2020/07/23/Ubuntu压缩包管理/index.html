Ubuntu压缩包管理

gzip

分别压缩，生成多个压缩包(.gz)，不保留源文件,只能压缩文件不能压缩目录

bzip2

分别压缩，生成多个压缩包(.bz2)，默认不保留源文件，只能压缩文件不能压缩目录

tar

压缩

参数:

c –创建(压缩时使用)

x –释放(解压缩时使用)

v –显示提示信息

f –指定压缩文件的名字

z –使用gzip的方式压缩文件 .gz

j –使用bzip2的方式压缩文件 .bz2

注意：不使用z/j参数，该命令只能对文件或目录打包
